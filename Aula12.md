	### 	Switch - case

Parece muito com o comando If porém com este você consegue saber os possíveis valores da variável

```html
switch(expressão){
	case valor 1:
		//comandos
		break;
	case valor 2:
		//comandos
		break;
	case valor 3:
		//comandos
		break;
	default:
		//comandos
}
```

**break** = muda de um case para outro
